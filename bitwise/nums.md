# Bitwise AND of Numbers Range

URL: https://leetcode.com/problems/bitwise-and-of-numbers-range/

## Description

Given a range `[m, n]` where `0 <= m <= n <= 2147483647`, return the bitwise AND of all numbers in this range, inclusive.

## Examples

### Example 1

```
Input: [5,7]
Output: 4
```

### Example 2

```
Input: [0,1]
Output: 0
```
